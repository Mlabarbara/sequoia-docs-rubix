# Dell PowerEdge R440 Overview

The Dell PowerEdge R440 is a primary server in our infrastructure, serving as one of our main hypervisor hosts. This documentation covers the physical hardware specifications and configuration. For information about the Proxmox Virtual Environment installation and configuration, please see the Proxmox R440 documentation.

- **Location**: Server rack, middle shelf
